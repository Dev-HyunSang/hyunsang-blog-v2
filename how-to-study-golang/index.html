Go언어는 어디에 사용되나요?

Go언어는 다양한 곳에 생각보다 사용됩니다. 채용 공고와 다양한 글들, 오픈소스 프로젝트를 통해서 여러분들이 어떻게 사용되는지 짐작할 수 있게 작성해 보았습니다. 생각보다 다양한 곳에 Go언어를 사용하고 있어요!

Tucker의 Go언어 프로그래밍

회사에 들어오기 전에, 대표님께서 Go언어를 공부하라고 책을 선물해주셨습니다. 그 때 읽으면서 아주 많은 도움을 받았습니다.

책을 읽기 전에도 Tucker님의 유튜브 채널를 보면서 Go언어를 통해서 벡엔드 개발을 접할 수 있었습니다. Tucker Programming - Go 로 만드는 웹 강의를 보면서 많은 도움이 되었습니다.

본격적으로 Go언어 개발 입문하기

Go언어를 통해서 백엔드 개발을 하기 전에 Go언어의 백엔드 프레임워크의 상태계를 알아봐야 해요!

Top Go Web Frameworks를 확인하면 각 프레임워크마다 몇 개의 Star를 받았는지 알 수 있습니다.

추천드리고 싶은 프레임워크

제 경험을 토대로 추천 드립니다. 자신이 생각했을 때

Back-End

net/http
Go언어에서 기본적으로 제공하고 있는 패키지입니다. 기본적으로 백엔드 공부를 하기 위해서는 필수적으로 기본적으로 공부하기 좋은 패키지이자 프레임워크 입니다.

gofiber/fiber
Go언에서 다양한 프레임워크가 있지만 개인적으로 Node.js Express 문법과 비슷한 gofiber를 추천드립니다.
문법도 간단하고 예전에는 기능이 많이 없었지만 조금씩 업데이트 되면서 많은 기능들이 추가되었습니다.
저는 개인적으로 제일 좋아하고, 제일 많이 사용하는 프레임워크입니다!

ORM(Object Relational Mapping, 객체 관계 매핑)

ORM(Object Relational Mapping, 객체 관계 매핑)은 객체와 테이블을 자동으로 매핑해주는 기술입니다.

gorm.io/gorm
제가 예전에는 좋아했지만 현재는 자주 안 사용하고 있는 프레임워크입니다. 구조체를 사용해서 선언해 주면서 따로 omitempty 태그를 사용하여 선언해야함으로 생각보다 복잡하고 코드가 늘어난다는 단점이 있었습니다.
하지만 장점도 있습니다. 명확하게 이 구조체의 필드가 무엇을 뜻하는지, 그리고 어떻게 데이터베이스에 적용되는지를 알 수 있다는 점과 더불어서 이미 SQL 쿼리로 생성한 데이터베이스에 조금 더 쉽게 적용시킬 수 있다는 점이 좋았습니다 :)

entgo/ent with Facebook Open Source
Facebook(Meta)에서 오픈소스 프로젝트로 진행하는 ORM 프레임워크인 entgo는 회사 동료 분께서 사용하셔서 우연히 살펴보다가 매력적이라서 사용하고 있습니다. 데이터베이스에 대한 많은 공부가 되어 있어야지 사용할 수 있는 프레임워크인 듯 합니다!
